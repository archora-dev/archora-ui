# Badge

`ArchBadge` labels compact status, category, or metadata.

```vue
<script setup lang="ts">
import { ArchBadge } from "@archora/ui";
import "@archora/ui/styles.css";
</script>

<template>
  <ArchBadge variant="success"> Live </ArchBadge>
</template>
```

## Variants

`neutral`, `primary`, `success`, `warning`, `danger`, and `info`.
