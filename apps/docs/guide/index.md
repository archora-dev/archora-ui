# Introduction

**Archora UI** is a modern design system inspired by organic forms and bionic design principles. Professional tools should feel precise, calm, and alive without becoming rigid or dull.

## What Is Organic Flow Design?

Archora UI's organic flow style is built on a few principles:

1. **Soft forms:** Rounded shapes and smooth contours replace hard edges.
2. **Depth and layering:** Glass surfaces and subtle shadows create spatial hierarchy.
3. **Fluid interaction:** Motion feels responsive and lightweight.
4. **Light accents:** Focused glow and gradients highlight important states.

## Core Capabilities

- **Vue 3 + Composition API:** Built on the modern Vue stack.
- **TypeScript:** Fully typed components and props.
- **Accessibility:** WAI-ARIA patterns and keyboard support are built in.
- **Themes:** Deep dark-theme support with a light theme available.
- **Design tokens:** Flexible customization through CSS variables.

## Library Structure

- **@archora/ui:** Main component package.
- **@archora/tokens:** Design token source for colors, radius, shadows, spacing, and motion.
